Abstract: We investigated the neural correlates of working memory using fMRI. Thirty healthy participants (mean age 24.5, SD 3.2) were scanned at 3T while performing an n-back task. Images were preprocessed using SPM12 and analyzed with a general linear model. Results showed significantly increased activation in the dorsolateral prefrontal cortex during 2-back compared to 0-back (p < 0.001, FWE-corrected). Additionally, working memory performance was positively correlated with dorsal anterior cingulate activation (r = 0.45, p = 0.01).
